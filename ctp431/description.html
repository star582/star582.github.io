Description of Final Project

CTP431 20130181 Chansoo Kim

1. Intro

It is the 'random-lauchpad pattern generator, by input a midi file.

It use Tone.js, Tone.js.MidiConvert, and jquery external library.

Tone.js is used to make synth sound, and Tone.js.Midiconvert is used to parse midi file.

2. Description

1) Load a midi file

Load a midi file clicking drop-down field, or simple drop the file to the field.

2) Play

When 'Start' button is enbled, you can press button to play whole midi file. Launchpad will automatically pressed to make the sound.

3) Click pads

You can hear the sound of each pad by clicking the pad.

3. Concepts and Technical things

The key concept of this project is 'Bind each tones technically, and show it beautifully.

Basic simple method is binding every single 'tones' to the each pad of launchpad.
