1. Intro

It is the 'random-lauchpad pattern generator, by input a midi file.

It use Tone.js, Tone.js.MidiConvert, and jquery external library.

Tone.js is used to make synth sound, and Tone.js.Midiconvert is used to parse midi file.

2. Description

1) Load a midi file

Load a midi file clicking drop-down field, or simple drop the file to the field.

2) Play

When 'Start' button is enbled, you can press button to play whole midi file. Launchpad will automatically pressed to make the sound.

3) Track control

When you load a midi file complete, track-controllers are generates on the right side of Launchpad.

You can change the track color of pads by input RGB color code on the text field.

Also, you can choose volume, and synth type from the 4 presets.

4. Notes

1) When you load a file once, you have to refresh the webpage to load other file.

2) Monosynth, AMsynth, and FMsynth can't create chord sound (it can't generate multiple tones at same time)

3) When the playing sound is awkward, please refresh the site again.
